## ✨ **Key Features**

✅ **Fully Responsive** on all devices  
✅ **Smooth Scrolling** navigation  
✅ **Hero Section** with call-to-action  
✅ **Beautiful Coffee Menu** with items and prices  
✅ **Image Gallery** for showcasing drinks
